GyroBall

Игра | Аркада | Mobile

Игра для проверки удобства использования механики акселерометра. Игра состоит из уровней-лабиринтов с препятствиями, через которые необходимо прокатить шар. Управление — наклонами телефона. Для шарика доступно 4 различные механики, соответствующие стихиям: стихия 1, 2, 3, 4. Локации соответствуют выбранной стихийной тематике.
